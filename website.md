# Client Coaching Website

## SEO

### On-Page SEO

Commencing Phase 1 of the SEO Strategy: On-Page Optimization. This phase involves establishing a clear 'Information Architecture' by using semantic HTML (H1-H3 hierarchy) and descriptive meta-data. By defining unique titles and descriptions for every page, I am providing search engines with the necessary context to index the site accurately for keywords like 'Executive Coaching' and 'Leadership Strategy'.


Refining Phase 1 SEO by eliminating 'Heading Conflicts' and authoring individualized metadata. By ensuring each page has a unique Title and Meta Description, I am preventing duplicate content issues. Furthermore, correcting the Header hierarchy (limiting to one H1 per page) ensures that search engine crawlers accurately identify the primary topic of each specific page.

Finalizing Phase 1 On-Page SEO and Technical Directives. I authored individualized metadata for Dr. Vives's biography page to capture 'Personal Branding' search intent and implemented a `robots.txt` file. This technical file acts as the primary gatekeeper for search engine crawlers, ensuring efficient indexing and protecting the site's directory integrity.


<u>**Skills Applied**</u>
* **Crawler Management:** Learning how to communicate directly with search engine infrastructure via root-level text files.
* **Brand-Specific SEO:** Tailoring metadata to highlight a specific person's credentials (bilingualism, dual-doctorates) to match niche search queries.


### Robots.txt
Implementing the robots.txt protocol to manage search engine crawler behavior. This file acts as the primary 'gatekeeper' for the NLCS web server, providing explicit 'Allow' directives for public content and a direct reference to the XML sitemap. This ensures that Google and other search engines can index the site’s professional coaching content efficiently and accurately.

Refining Technical SEO by correcting the Sitemap directive in the robots.txt file. Initially, the file contained a search-engine query rather than a direct path. By providing a direct URL to sitemap.xml, I am ensuring that search engine crawlers can ingest the site's architecture immediately and without errors, which is a vital step for rapid indexing of the new domain.

<u>**Skills Applied**</u>
* **Crawler Management:** Mastering the syntax used to communicate directly with web robots and spiders.
* **Server Architecture:** Understanding the 'Root Directory' structure and how global configuration files affect site-wide visibility.
* **Protocol Standards:** Implementing industry-standard SEO directives that differentiate a professional production site from a development prototype.
